      • 과제 관련
        • 매주차 선정된 쿠키즈 깃헙과 비교/분석
          • 코멘트와 코드를 보며 나랑 어떻게 다른지 확인
        • 내 코드들의 문제점과 차이점 모두 정리해서 노션 페이지에 정리
        • 각 주차와 다음 주차에 달린 맛도리 질문 답변 정독
          • 정리된 내 노션과 겹치는 부분 탐색, 새로운 확장성 확인
      • 카테캠 꿀팁 링크
        • 꿀팁 링크에 들어가 내용 확인 후 대주제로 링크끼리 분류
        • 멘토님이 주신 링크에 들어가 해당 내용을 정독 및 메인 페이지 자료 확인
          • 첨부된 모든 링크의 내용을 다 읽어보고 각 링크가 얘기하고자 하는 내용 이해
          • 각 링크 별 자신의 리뷰 작성
      • 최종
        • 최종적으로 정리된 내용과 과제 피드백 토대로 다시 만들어본다
        • 링크들을 통해 얻은 인사이트로 새로운 부분에 대해 공부한다

    슬랙이나 컨퍼런스에서 느끼는 자극들을 최대한 많이 느끼고 기록하고 소화하자
